Medicine Detail Tracker

Overview

The Medicine Detail Tracker is a Java application built using Swing for managing medicine records, including adding, viewing, checking expiration, and deleting medicines.

Features

  • Add new medicines with their expiration dates.
  • View all added medicines with their expiration dates.
  • Check for medicines expiring within a specified number of days.
  • Delete medicines from the tracker.
  • User-friendly GUI interface using Swing components.

Installation

  1. Download the Java source code files.
  2. Ensure you have Java Development Kit (JDK) installed on your system.
  3. Download the JDatePicker-1.3.4.jar library.
  4. Compile the Java code using the following command: 
    java -cp lib/JDatePicker-1.3.4.jar MedicineDetailTrackerUI.java
  5. Run the application using: 
    java -cp .:lib/JDatePicker-1.3.4.jar MedicineDetailTrackerUI

Usage

To use the Medicine Detail Tracker:

  1. Upon running the application, you will see a menu with options to Add Medicine, Check Expiring Medicines, View All Medicines, and Delete Medicine.
  2. Click on each menu option to navigate through different functionalities of the application.
  3. Follow the on-screen instructions to add, view, check, or delete medicines.
